About this project

A License plate is the unique most important identifier of a vehicle .The aim of this academic project is license Plate detection using state of the art computer vision and deep learning techniques.

The main 3 parts of the project are :

License Plate Detection



YoloV5m

License Pate Segmentation



OpenCV

Optical Character Recognition


Tesseract
